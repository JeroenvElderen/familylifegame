# Project Guardrails

This project is a private browser-based life simulation game built with React (Vite).

## Core rules
- Every 1 second `incomePerSecond` modifies money (can be positive or negative).
- Every 30 seconds = 1 day.
- Every 365 days = +1 year age.
- Every 30 seconds a random event appears with 4 choices.

## Product requirements
- Stay browser-only (no backend).
- Be private/local.
- Use clean, scalable architecture.
- Avoid unnecessary complexity.
- Be easy to extend with more events later.

## Engineering note
